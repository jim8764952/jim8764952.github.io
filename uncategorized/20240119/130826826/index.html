為甚麼你應該學Python ?

Posted on 2024-01-19 Edited on 2024-01-31 Views:
透過Python 能快速掌握程式的精隨透過，再透過知識遷移學習更深的程式碼。

相信大家在國小的時候有學過 Scratch ，他可以用拖
放 (drag and drop) 的方式去建立你的程式碼，一節上課就能有小作品出來，那時的我是無比開心，這就是成就感，當學習一件事情時有成就感才能推動你繼續學習。

時間到了高中，想說延續寫程式的熱情那報名 APCS大學程式設計先修檢測來看我的能力在哪裡，當時我選擇 c 語言作為我準備的方向，想說就看看線上影片學習就好，但試後成績真的是慘不忍睹。過了一段時間後，發現那時學習時遇到很多挫折多半是卡在 C 語言語法上很容易一不小心就錯，而且也沒有報名相關課程，相信還是有很多厲害的人，學 C 語言很快就上手。後來接觸到 Python 當我有點子時能很快到利用( Import )別人寫好的模組包去實作我的想法像是，小遊戲、抓取股票網站上的資訊、資料視覺化。

總結一下，Python 能在前期訓練邏輯或是操作演算法，如果想要延伸能繼續往 Python底層語言 C 語言學習，因為他能提供更好的效能。
