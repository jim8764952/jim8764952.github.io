Hexo加入留言功能吧! giscus安裝教學 Next主題

Posted on 2024-01-04 Edited on 2024-01-06 Views:
這篇文章將介紹如何在 Hexo blog中安裝 Giscus 評論系統。Giscus 是一個開源的、基於 GitHub 的評論系統，具有強大的功能和靈活的配置。

giscus 介紹

我在安裝giscus前已經有試過utterances好一陣子，但都找不到問題點，想說網站剛起步就直接嘗試其他留言插件試試，不試還好一試就成功。

相較於utterances使用GitHub Issue來作為存放留言的地方，giscus是使用GitHub Discussions。作者提到是受utterances啟發而開發的，相關步驟跟utterances極為類似，那我們就往下看怎麼安裝吧!

相關版本供參考:

  • hexo: 7.0.0
  • hexo-cli: 4.3.1
  • hexo-next-giscus@1.2.0
  • hexo-theme-next@8.19.1
  • hexo@7.0.0

安裝

  1. 將repo設為公開
  2. 安裝**giscus**
  3. 將儲存庫的Discussions功能打開
  4. 安裝套件
    npm install hexo-next-giscus --save
  5. 在_config.next.yml填入以下(我是寫在Comments Settings下方未來要設定比較好找)
    # giscus
# For more information: https://github.com/next-theme/hexo-next-giscus
giscus:
	enable: true
    repo: name/name.github.io # Github repository name
    repo_id: # Github repository id
    category: Announcements # Github discussion category
    category_id: DIC_kwDOK7APPM4CcMQg # Github discussion category id
    # Available values: pathname | url | title | og:title
	mapping: pathname
  	# Available values: 0 | 1
	reactions_enabled: 1
  	# Available values: 0 | 1
  	emit_metadata: 0
  	# Available values: light | dark | dark_high_contrast | 		transparent_dark | preferred-color-scheme
  	theme: noborder_dark
  	# Available values: en | zh-CN
  	lang: zh-TW
  	# Place the comment box above the comments
  	input_position: bottom
  	# Load the comments lazily
	loading: lazy
    這邊的參數調整可以在giscus設定填入你的repo、主題及語言等，下方就會得到一段Script(如下圖)，照著參數名稱修改即可。
